# 에디의 하루 (Eddie's Day Planner)

ADHD 사용자의 **계획(캘린더) → 실행(루틴 동행)** 을 하나로 잇는 동행형 플래너 앱 시안.
캐릭터 '에디'가 시간맹·미루기·망각을 대신 관리해주는 컨셉.

> `mungbws1031/ivdr-wiki` 저장소에서 시안 검증 후 별도 브랜드로 분리된 독립 프로젝트입니다.

## 지금 구현된 것 — 하단 4탭 전체

### 오늘 탭 (실행 뷰)
- **에디 시계** — 실제보다 5~10분 빠른 표시(오프셋 비노출), 감정 4종 마스코트
- **출발 카운트다운** — 준비+이동 역산, 여유/임박 상태, 나가기 전 체크
- **오늘 타임라인** — 예정·진행·완료·놓침·이월·패스 상태(색+아이콘+라벨 병행)
- **할 일 → 타임블록 제안 모달** — 빈 시간 제안, 수락해야만 배치(자동 배치 금지)
- **실패·회복 모달** — 놓침 시 자책 없는 회복 경로(지금/저녁/내일/패스), 스트릭 리셋 없음
- **복약 카드** — 원탭 확인, 비의료기기 면책 문구

### 캘린더 탭 (계획 뷰)
- **주 뷰** — 요일 스트립 + 선택일 아젠다, 빈 날은 안심 톤 empty state
- **월 뷰** — 상태색 도트로 밀도 표시, 날짜 탭하면 주 뷰로 드릴다운
- **일정 추가** — 제목/시작/소요시간/카테고리/준비+이동시간 반영, 3탭 이내, 겹침 비차단 경고

### 에디 탭 (동행·동기부여)
- **캐릭터 상태 + 격려** — 담백한 문구, 과장 없음
- **회복형 스트릭** — 7일 스트립, '리셋' 대신 '회복' 개념
- **저녁 회고** — 핵심 1개 이상 완수 시 '괜찮은 하루', 내일 미리보기, 굿나잇

### 더보기 탭 (관리·설정)
- **루틴 빌더** — 아침/저녁 단계 추가·on-off
- **복약 설정** — 약 추가/삭제 + 비의료기기 면책 문구
- **제자리 맵** — 자주 잃는 물건 → 지정 위치
- **데이터·프라이버시** — 민감정보 동의, 내보내기, 전체 삭제(2단계 확인)
- **알림 설정** — 알림 종류별 on/off, 방해금지 시간대, 알림 피로 방지 안내

빠른 캡처 FAB은 모든 탭 우하단에 상시 노출됩니다.

## 금지 패턴 (반드시 지킬 것)

- 자책 유발 문구 절대 금지 — 놓침·이월은 담백/회복 톤
- 빠른 시계의 정확한 오프셋 노출 금지
- 자동 일정 배치 금지 — 항상 "제안 후 수락"
- 정서적 과의존 유도 카피 금지

## 개발

```bash
npm install
npm run dev       # http://localhost:5173
npm run build      # 타입체크 + 프로덕션 빌드
```

390×844(iPhone 14) 모바일 기준. 데스크톱 넓은 화면에선 자동으로 폰 프레임에 감싸져 표시됩니다.
